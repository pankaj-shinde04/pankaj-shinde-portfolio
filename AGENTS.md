## Project Summary
A modern personal portfolio website for Pankaj Dnyaneshwar Shinde, a Full Stack Developer. The site features a clean, responsive design with smooth scroll animations, a hero section, about, skills, projects, experience, and contact sections. It clones the visual style and aesthetic of shubhporwal.me.

## Tech Stack
- Framework: Next.js 15+ (App Router)
- Styling: Tailwind CSS 4
- Icons: Lucide React
- Deployment: Vercel (recommended)

## Architecture
- `src/components/sections/`: Contains individual section components (Hero, About, Skills, etc.)
- `src/app/page.tsx`: Main entry point assembling all sections.
- `src/app/globals.css`: Global styles and design tokens.

## User Preferences
- Theme: Dark theme by default (pure black background).
- Visual Style: Glassmorphism, high-contrast typography, minimalist iconography.
- Responsiveness: Fully responsive for mobile, tablet, and desktop.
- Animations: Smooth scroll and subtle entrance animations.

## Project Guidelines
- Follow the visual style of shubhporwal.me.
- Use Pankaj's resume data for content.
- Maintain a clean and professional UI.

## Common Patterns
- Section-based architecture.
- Tailwind CSS for styling.
- Framer Motion or CSS animations for visual effects.
